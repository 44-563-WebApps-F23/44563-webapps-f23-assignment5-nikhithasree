Cities row breaks first

As I reduce the browser size, the city names breaks first because they are in columns with the "col-md" class. Bootstrap's responsive design adjusts the column sizes based on the screen width,so "col-md" class columns will stack vertically on smaller screens whereas "col-sm-2" class columns will remain in a row longer.
